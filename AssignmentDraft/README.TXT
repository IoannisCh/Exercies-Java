AUTHOR: Ioannis
USER INSTRUCTIONS: Start using this application by right click on the 
TestListString class and chose the method you need to use. The system 
will produce and print the result on the BlueJ console for you. Then 
if more results needed, select clear from the options, as the console 
will keep the previous results.
